# Devtech

Interni frontend prototip baziran na React + Vite + MUI.

## Pokretanje projekta

- Instalacija:
  - `npm install`
- Dev server:
  - `npm run dev`
- Lint:
  - `npm run lint`
- Build:
  - `npm run build`
